# Required Assets

The following assets need to be added to the `public` directory:

## Profile Images
- Profile picture: `/public/profile.jpg` (or update the path in `containers/PortfolioContainer.tsx`)

## Brand Design Images
- Brand design examples: Add images to `/public/brand-design/` directory
- Update the `brandDesignImages` prop in `PortfolioContainer.tsx` with the actual image paths

## Notes
- All images should be optimized for web use
- Recommended formats: WebP, PNG, or JPG
- Profile image should be square (recommended: 160x160px or larger)
